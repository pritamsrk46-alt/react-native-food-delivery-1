# 🍔 Food Delivery App

## 📌 About Project
This is a React Native Food Delivery App.

## 🚀 How to Run
git clone https://github.com/pritamsharmareactnative/react-native-food-delivery.git
cd react-native-food-delivery
npm install
npx react-native run-android

## 📱 Requirements
- Node.js
- Android Studio
